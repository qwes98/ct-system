# Project Milestones

MVP 개발 마일스톤 및 릴리즈 계획입니다.

## 1. MVP 개요

### 1.1 프로젝트 목표

**핵심 목표:**
- 코딩테스트 연습 플랫폼 MVP 출시
- 문제 풀이 및 제출 기능 구현
- Judge0 기반 코드 실행 환경 구축

**비용 목표:** 월 1만원 이하 (~$7-8 USD)

**성공 지표:**
| 지표 | 목표 |
|------|------|
| 제출 완료율 | 90%+ |
| Submit 응답 시간 | P95 < 10초 |
| API 가용성 | 95%+ (MVP 완화된 목표) |

### 1.2 MVP 범위

**포함:**
- 문제 리스트/상세 페이지
- 코드 에디터 (4개 언어)
- Run (샘플 테스트)
- Submit (전체 테스트)
- 게스트 제출 이력

**제외:**
- 사용자 인증
- 리더보드/랭킹
- 부분 점수
- 상세 에러 로그

## 12. 의사결정 로그

| 날짜 | 결정 사항 | 근거 |
|------|-----------|------|
| - | 게스트 only MVP | 빠른 출시, 인증 복잡도 제외 |
| - | Judge0 self-hosted | 커스터마이징, 비용 절감 |
| - | Vercel + Oracle Cloud/VPS | 프론트 최적화, 월 1만원 이하 비용 |
| - | SQLite (MVP) | 단일 서버 구성, 설치 불필요, 확장시 PostgreSQL |
| - | In-Memory Queue (MVP) | Redis 불필요, 저사양 VM 최적화 |
| - | 동시성 축소 (Run 10, Submit 5) | MVP 단계 유저 수 적음 |
